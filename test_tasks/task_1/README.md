## Задание № 1
> Почтальон должен обойти несколько пунктов и вернуться на работу (в исходную точку).
> Заданы точки обхода с указанием координаты точки.
> - Почтовое отделение (0, 2)
> - Ул. Грибоедова (2, 5)
> - Ул. Бейкер стрит (5, 2)
> - Ул. Большая Садовая (6, 6)
> - Вечнозелёная Аллея (8, 3)
>
> Написать функциональным методом программу, которая расчитывает полным перебором
> оптимальный маршрут и возвращает в консоль строку вида:
> 
> (0, 3)[5.115551275463989] --> ... --> (2, 0)[2.325661275463977] = 25.612754639771155

## Решение
Типичная "задача коммивояжера", не имеющая алгоритмического решения. Два способа решения:

1) методом полного перебора (по условию);
2) методом Литтла.
